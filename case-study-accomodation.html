CASE STUDY 02

Accommodation Extractor

Automated PDF parsing system for structured accommodation record management.

The Problem

Accommodation booking and usage records were submitted as PDFs and required manual transcription into operational spreadsheets. This introduced delays, inconsistencies, and risk of data entry errors.

System Architecture

PDF SUBMISSION
↓
FILE INGESTION
↓
PYTHON PARSER
↓
FIELD EXTRACTION LOGIC
↓
STRUCTURED DATA OUTPUT
↓
EXCEL SHEET AMENDMENT

My Role

Designed and implemented PDF parsing logic to reliably extract accommodation data fields. Structured the extraction rules to handle formatting variation while preserving data integrity.

Technical Stack

Python · PDF Parsing · Data Normalisation · Excel Automation

Outcome

Significantly reduced manual administrative work and introduced a repeatable structured extraction workflow for accommodation records.
